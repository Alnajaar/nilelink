# Menu Management Guide

Learn how to create, update, and manage your restaurant menu on NileLink.

## Menu Structure

### Categories
Organize your menu into logical categories:
- Appetizers
- Main Courses
- Desserts
- Beverages
- Dietary Options (Vegetarian, Vegan, Gluten-Free)

### Menu Items
Each item should include:
- Name
- Description
- Price
- Preparation time
- Dietary information
- High-quality photo

## Adding Menu Categories

1. Go to your Dashboard
2. Click "Menu" > "Categories"
3. Click "Add Category"
4. Enter category details:
   - Name (e.g., "Main Courses")
   - Description (optional)
   - Display order
   - Availability hours

## Adding Menu Items

### Basic Information
1. Navigate to "Menu" > "Items"
2. Click "Add Item"
3. Fill in details:
   - Item name
   - Category selection
   - Detailed description
   - Base price

### Advanced Options
- **Variations**: Size options (Small/Medium/Large)
- **Add-ons**: Extra ingredients or customizations
- **Modifiers**: Required choices (e.g., spice level)
- **Dietary Tags**: Vegetarian, Vegan, Gluten-Free, etc.

### Image Upload
- Use high-resolution images (minimum 800x600px)
- Show the actual dish
- Good lighting and presentation
- PNG or JPEG format
- File size under 5MB

## Special Offers

### Creating Promotions
1. Go to "Marketing" > "Promotions"
2. Click "Create Promotion"
3. Choose promotion type:
   - Percentage discount
   - Fixed amount off
   - Buy one get one
   - Free delivery
