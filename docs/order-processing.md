# Order Processing Guide

Master the art of handling orders efficiently on NileLink. This guide covers everything from receiving orders to delivery completion.

## Order Flow Overview

1. **Order Received** - New order notification
2. **Accept/Reject** - Within 2 minutes
3. **Preparation** - Kitchen workflow
4. **Ready for Pickup** - Order completed
5. **Pickup** - Driver collects order
6. **Delivery** - Customer receives order
7. **Completion** - Order closed

## Receiving Orders

### Order Details

### Auto-Accept Settings

  - Peak hours

  - Small orders

## Order Management

### Accepting Orders
1. Review order details
2. Check inventory availability
3. Confirm preparation capacity
4. Accept within 2 minutes
5. Send confirmation to customer

### Rejecting Orders
- Only reject for valid reasons:
  - Item unavailability
  - Kitchen overload
  - Delivery area issues
- Provide clear rejection reason
- Suggest alternatives when possible

### Order Modifications
- Handle customer requests
- Update item quantities
- Add/remove items
- Adjust delivery time
- Update special instructions
