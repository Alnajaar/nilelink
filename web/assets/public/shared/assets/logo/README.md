# NileLink Brand Assets

## Logo Files
- `logo.png` - Primary logo (light background)
- `logo-white.png` - Logo for dark backgrounds
- `logo-square.png` - Square logo variant
- `logo-icon.ico` - Favicon/Icon version

## Usage Guidelines
- Use `logo.png` for light backgrounds
- Use `logo-white.png` for dark backgrounds and navigation
- Use `logo-square.png` for app icons and avatars
- Use `logo-icon.ico` as favicon

## Color Palette
- Primary: `#0A2540` (Deep Navy Blue)
- Secondary: `#00C389` (Emerald Teal)
- Accent: `#F5B301` (Soft Gold)
